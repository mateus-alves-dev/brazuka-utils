# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | ✅        |

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Use [GitHub Private Vulnerability Reporting](https://github.com/mateus-alves-dev/brazuka-utils/security/advisories/new) to report a vulnerability confidentially.

Include as much of the following information as possible:

- Type of vulnerability (e.g. XSS, injection, incorrect validation logic)
- Affected version(s)
- Step-by-step instructions to reproduce the issue
- Potential impact

You will receive a response within **72 hours**. If the issue is confirmed, a patch will be released as soon as possible.
